# Job Board Backend

Welcome to the **Job Board Backend**! This repository hosts the backend API for managing job listings and applications.

## Features
- User authentication and authorization
- Job posting and management
- Application tracking

## Getting Started

### Prerequisites
Before you begin, ensure you have met the following requirements:
- [Node.js](https://nodejs.org/en/) installed
- A MongoDB database set up

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/EinsteinDipondo/job-board-backend.git
   cd job-board-backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application
To start the server, run:
```bash
npm start
```
